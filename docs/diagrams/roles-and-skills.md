  - DevOps Engineer
  - Data Architect

## Roles and responsibilities

1. **Backend Engineer**
    Develops server-side logic, creates APIs (REST/gRPC) for data exchange, and integrates with databases and external services.
2. **Site Reliability Engineer (SRE)**
    Ensures system availability and scalability, manages cloud infrastructure (Kubernetes), and configures monitoring and alerting systems.
3. **Database Administrator (DBA)**
    Manages database performance, handles sharding and replication, optimizes SQL queries, and ensures data backups and recovery.
4. **QA Automation Engineer**
    Writes scripts to automatically test software features, performs load testing, and ensures new code doesn't break existing functionality.
5. **Systems Analyst**
    Translates business requirements into technical specifications, designs data flows, and documents system behaviors for developers.

## Common skills across roles

- **Version Control:** Git.
- **Operating Systems:** Linux/Unix terminal usage.
- **Containerization:** Docker, basic Kubernetes concepts.
- **Data Handling:** SQL, JSON/XML formats.
- **Communication:** Understanding HTTP, API principles (REST), and technical documentation.
